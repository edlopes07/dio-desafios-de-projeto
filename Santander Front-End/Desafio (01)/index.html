Criando Tabelas com HTML

(voltar)

O QUE APRENDI:* Nas aulas sobre tabelas em HTML, aprendi como organizar dados de um jeito que fique fácil de entender na página. Com as tags certas, dá pra criar aquelas “grades” que a gente vê em planilhas ou listas, dividindo as informações em linhas e colunas. É bem útil quando você quer mostrar vários dados juntos, tipo horários, preços ou estatísticas. Também vi que dá para destacar os títulos das colunas, juntar células quando precisar, e até deixar tudo com um visual mais arrumadinho usando atributos simples. Agora sei que as tabelas são uma ferramenta prática para deixar o conteúdo mais claro e organizado.

Entendendo HTML Semântico

(voltar)

O QUE APRENDI:* Na aula sobre HTML semântico, aprendi a usar tags que dão significado ao conteúdo da página, como header, main, footer, aside, blockquote, figure e figcaption. Também entendi o uso do picture para carregar imagens diferentes conforme o dispositivo. Essa organização ajuda tanto na acessibilidade quanto no web scraping, facilitando que programas capturem informações do site de forma eficiente. Usar HTML semântico deixa o site mais claro, acessível e otimizado.
